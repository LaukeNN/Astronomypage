3 comandos, para guardar y subir al repositorio, dejar en linea el cheack point.

1- git add . (agrega todo el punto es para todos los archivos)
2- git commit -m "escribes un msj" (es una etiqueta para el nombre que va a tener la version de guardado que subiste)
